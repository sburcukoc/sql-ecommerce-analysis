# SQL E-commerce Data Analysis

This project demonstrates basic SQL database design and data analysis.

## Project Overview
The goal of this project is to create a simple e-commerce database and analyze customer spending using SQL queries.

## Tables
customers
- id
- name
- country

orders
- order_id
- customer_id
- amount

## Files
schema.sql → creates database tables  
data.sql → inserts sample data into the tables  
analysis.sql → SQL queries for data analysis

## Analysis
The analysis queries show:
- total spending per customer
- the top spending customer
- the average order value
